# Olá ao Cyni Store
<br>
pra você fazer um app você precisa de:


* info.json
* icon.png
* application.apk


código de exemplo pra Info.json
```json
{
 "Label": "NomeDoSeuApp",
 "Owner": "Nomepravocecolocar",
 "Visibility": "Public", // ou até Private
 "Description": "Descrição aqui" // Opcional 
 "Icon": "icon.png"
 "Install": "application.apk"
}
```
